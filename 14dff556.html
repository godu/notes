Successor

  1. ∀ a ∈ ℕ ⇒ S(a) ∈ ℕ
  2. ∀ a,b ∈ ℕ ⇒ a = b ∧ S(a) = S(b)
  3. ∀ a ∈ ℕ ⇒ S(a) != 0

Addition

  • ∀ a ∈ ℕ ⇒ a + 0 = a
  • ∀ a,b ∈ ℕ ⇒ a + S(b) = S(a + b)
(+) :: Natural -> Natural -> Natural
a + Zero = a
a + (Succ b) = Succ $ a + b

Multiplication

  • ∀ a ∈ ℕ ⇒ a * 0 = 0
  • ∀ a,b ∈ ℕ ⇒ a * S(b) = a + (a * b)
(*) :: Natural -> Natural -> Natural
a * Zero = Zero
a * (Succ b) = a + (a + b)
